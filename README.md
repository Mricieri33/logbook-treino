# Logbook de Treino

App de registro de treino (semanas/meses) com análise de evolução de reps e carga. PWA — instala na tela inicial e funciona offline.

## Publicar no GitHub Pages

1. Criar repositório novo (ex: `logbook-treino`)
2. Subir todos os arquivos desta pasta na raiz do repo:
   - `index.html`
   - `manifest.json`
   - `sw.js`
   - `icon-192.png`
   - `icon-512.png`
3. Settings → Pages → Branch: `main`, pasta `/ (root)` → Save
4. Acessar `https://mricieri33.github.io/logbook-treino`

## Instalar no celular

1. Abrir a URL no Chrome (Android)
2. Menu (3 pontos) → "Adicionar à tela inicial" / "Instalar app"

## Dados

- Salvos em `localStorage` do navegador (por dispositivo)
- Limpar dados do Chrome apaga o histórico do app
